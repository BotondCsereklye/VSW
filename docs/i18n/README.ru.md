# Руководство пользователя VSW

## Назначение

VSW является защитным локальным сканером для доменов и IP-адресов. Он проверяет security headers, TLS-информацию, небольшой безопасный список портов и выбранные read-only признаки неправильной конфигурации.

## Запуск

Рекомендуемый вариант для Windows:

```powershell
.\launch_vsw_launcher.ps1
```

Альтернатива через Docker:

```bash
docker compose up --build
```

## Использование

1. Откройте VSW app: `http://localhost:5173`.
2. Введите домен или IP, который вам разрешено проверять.
3. Просмотрите группы score, findings, историю, экспорт и link checks.
4. Browser extension используйте только как link-capture helper для локального backend.

## Browser Extension

- Обычные ссылки внутри страницы можно предварительно проверить перед переходом.
- Address bar и bookmark клики в Manifest V3 нельзя надёжно заблокировать до загрузки.
- Такие посещения пассивно записываются после загрузки через `webNavigation`.
- `Trust site` пропускает сканирование и блокировку для этого host.
- `Ignore minimum score` всё равно создаёт отчёты, но не блокирует только из-за score.

## Ограничения

Нет exploit-функций, нет brute force, нет агрессивного сканирования и нет попыток аутентификации.
